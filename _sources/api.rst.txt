API reference
===============

The impunity library is based on three main core objects:

- impunity wrapper accessible through :func:`impunity.wrapper.impunity`;
- An AST visitor defined in the class :class:`impunity.visitor.Visitor`;
- Representation of AST nodes with :class:`impunity.quantityNode`;
